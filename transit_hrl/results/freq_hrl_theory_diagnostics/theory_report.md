## Theorems

### T1: Causal Encoder Nonanticipativity

Statement: For any time t, frequency features z_t produced by a CausalSpectralEncoder are measurable with respect to the sigma-field generated by bins observed at times <= t.

Assumptions:
- The domain adapter only passes bins to encoder.update after their event time.
- Encoder state transitions read only the previous encoder state and the current bin.
- Feature calls do not run offline smoothing over future bins.

Proof sketch: Induct on update count. The initial state is deterministic. If the state before update k is a function only of bins 1..k-1, the next state is a deterministic function of that state and bin k. Therefore every feature emitted after update k is a function only of bins 1..k.

Diagnostics: test_causality covers EMA, Fourier, state-space, wavelet, adaptive wavelet, and neural/PINN state-space prefixes.

### T2: Leakage-Shaped Return Lower Bound

Statement: If leakage L_t >= 0 and shaped reward is r'_t = r_t - lambda L_t with lambda >= 0, then shaped episodic return is a conservative lower bound on task return up to lambda sum_t L_t.

Assumptions:
- Leakage is computed causally from upper and lower action effects.
- The leakage multiplier is nonnegative.
- The task return is the unshaped sum of r_t on the same trajectory.

Proof sketch: Summing r'_t gives sum_t r_t - lambda sum_t L_t. Since lambda >= 0 and L_t >= 0, shaped return cannot exceed task return. The exact gap is lambda sum_t L_t.

Diagnostics: paper statistical checks report lower-LF drift and no-tradeoff gates for Trading and Transit constraints.

### T3: Primal-Dual Leakage Constraint Direction

Statement: For a fixed policy class and positive dual step size, the primal-dual update increases the leakage multiplier when observed constraint cost exceeds its budget and decreases it when the cost is below budget.

Assumptions:
- Constraint samples are finite and computed on the current rollout.
- The dual variable is projected onto a nonnegative bounded interval.
- The actor update includes the current multiplier times constraint excess.

Proof sketch: The dual update is projected gradient ascent on eta E[c_t - budget]. A positive excess gives a positive gradient for eta; a negative excess gives a negative gradient. Projection preserves nonnegativity and boundedness.

Diagnostics: DualActorCriticPPO tests plus trading/transit primal-dual validation check the expected drift direction.

### T4: Promotion Threshold Tradeoff

Statement: For a persistent residual detector with threshold tau and persistence ratio rho, increasing tau or rho weakly reduces stationary false positives but can increase detection delay and false negatives after true persistent shocks.

Assumptions:
- Promotion uses only residual magnitudes in a finite causal window.
- False positive means the window statistic crosses the gate in a stationary regime.
- True shock means the residual distribution shifts upward for multiple bins.

Proof sketch: The promotion event is a monotone threshold event in residual exceedance counts. Raising tau or rho can only remove crossing windows under the same residual path. That reduces stationary crossings, but it can also delay or remove crossings on a shifted path whose exceedance count is near the old threshold.

Diagnostics: promotion sweep, recovery validation, and learned Transit promotion-replan checks should be reported together.
